# Module E2E Structure

Each subfolder maps to one product module and includes:
- `hr-workflows.spec.ts` for HR user workflows
- `employee-workflows.spec.ts` for non-HR (employee) workflows

Modules covered:
- dashboard-home
- profile
- my-leave
- my-payslips
- performance-evaluations
- announcements-polls
- user-management
- shift-management
- user-attendance-management
- overtime-management
- leave-management
- salary-structure
- payslip-management
- payroll-settings
- reports-analytics
- app-logs
- shared-resources
